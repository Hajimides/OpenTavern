# Contributing to OpenTavern

Thank you for your interest in contributing to OpenTavern!  
We welcome contributions of all kinds, especially character cards.

## No CLA Required

We do **not** require contributors to sign a Contributor License Agreement (CLA).  
By submitting a Pull Request, you agree that your contribution is licensed under **AGPL-3.0** as part of this project.

## Our Commitment

- I have no plans to commercialize this project.
- I will not sell or commercially exploit character cards contributed by the community.
- Contributed character cards will remain part of the open-source project under AGPL-3.0.

## How to Contribute Character Cards

We especially welcome new character cards!

1. Place your character card JSON file inside the `cards/` folder.
2. You only need to add the JSON file. **Please do not modify `index.html`** or other core files.
3. If you only want to contribute character cards, there is no need to change any code.

> **Important**: Do not hardcode character card paths into `index.html`. The application is designed to load cards dynamically from the `cards/` folder.

## Code Contributions

If you want to improve the web application (HTML, JavaScript, UI, features, etc.), please follow these guidelines:

- Please discuss significant changes in an Issue first before submitting a large Pull Request.
- Keep changes focused. If your goal is only to add character cards, please do **not** modify `index.html`.
- Follow the existing code style as much as possible.

## Pull Request Guidelines

- Keep your Pull Request focused and small when possible.
- Clearly describe what your contribution does in the PR description.
- If you are only contributing character cards, simply mention it in the PR title or description.

## License

This project is licensed under the **GNU Affero General Public License v3.0 (AGPL-3.0)**.

Thank you again for contributing! Your support helps make OpenTavern better for everyone.

# 为 OpenTavern 做贡献

感谢你有兴趣为 OpenTavern 做出贡献！  
我们欢迎各种形式的贡献，尤其是角色卡的贡献。

## 无需签署 CLA

本项目**不要求**贡献者签署 Contributor License Agreement（CLA）。  
提交 Pull Request 即表示你同意将你的贡献以 **AGPL-3.0** 许可授权给本项目。

## 我们的承诺

- 我目前没有将本项目商业化的计划。
- 我不会将社区贡献的角色卡进行商业贩卖或商业利用。
- 贡献的角色卡将作为开源项目的一部分，继续受 AGPL-3.0 许可保护。

## 如何贡献角色卡

我们非常欢迎大家贡献新的角色卡！

1. 请将角色卡的 JSON 文件放在 `cards/` 文件夹内。
2. 你只需要添加 JSON 文件即可，**请不要修改 `index.html`** 或其他核心文件。
3. 如果你只是想贡献角色卡，完全不需要改动任何代码。

> **重要提醒**：请不要将角色卡路径硬编码到 `index.html` 中。本应用设计为从 `cards/` 文件夹动态加载角色卡。

## 代码贡献

如果你想改进网页应用（HTML、JavaScript、界面、功能等），请注意以下事项：

- 较大的改动建议先在 Issue 中讨论后再提交 Pull Request。
- 请保持修改聚焦。如果你只是想添加角色卡，请**不要修改 `index.html`**。
- 请尽量遵循现有的代码风格。

## Pull Request 提交规范

- 建议保持 Pull Request 专注且简洁。
- 请在 PR 描述中清楚说明你的贡献内容。
- 如果你只是贡献角色卡，请在 PR 标题或描述中注明。

## 许可证

本项目采用 **GNU Affero General Public License v3.0 (AGPL-3.0)** 许可。

再次感谢你的贡献！你的支持能让 OpenTavern 变得更好。
